## 🎯 主な変更点

### 1. **簡単な日本語に変更**
- ❌ 削除: 難しい言葉、専門用語、複雑な表現
- ✅ 追加: 中学生でもわかる簡単な日本語
- ✅ 短い文章で読みやすく

### 2. **励ましと認める言葉を最初に**
恋愛の例：
```
今まで本当によく頑張ってきましたね。
恋愛で悩むのは、あなたが本気で誰かを大切に思っているからです。
その気持ちは、とても純粋で素晴らしいものです。
あなたは愛される価値がある人です。
```

金運の例：
```
毎日本当にお疲れ様です。
お金のことで悩むのは、あなたが真面目に人生を考えているからです。
あなたはとても頑張っています。
その努力は、必ず報われます。
```

健康の例：
```
辛い中、よく耐えてきましたね。
体や心が辛いのに、それでも毎日を過ごしているあなたは、
本当に強い人です。あなたは十分頑張っています。
```

### 3. **新しい構成（5セクション）**

#### 【あなたへの言葉】（3-4行）
→ 相手の頑張りを認め、励ます

#### 【今のあなたの状態】（2-3行）
→ 今の気持ちに寄り添う

#### 【シャーマンからのアドバイス】（箇条書き3つ）
→ 具体的で簡単にできること

**例（恋愛）:**
- 毎朝、鏡を見て自分に笑顔を向けて「私は素敵です」と言ってみましょう
- 好きな人のことを考えながら、ピンク色のものを身近に置きましょう
- 友達との楽しい時間を大切にしましょう。笑顔でいると運気が上がります

**例（金運）:**
- 財布の中を整理して、きれいにしましょう。お金は綺麗な場所が好きです
- 毎日「ありがとう」と感謝の言葉を10回言いましょう。感謝は豊かさを呼びます
- 小さな節約でも良いので、続けましょう。小さな積み重ねが大きくなります

**例（健康）:**
- 深呼吸を朝と夜に5回ずつしましょう。ゆっくり吸って、ゆっくり吐きます
- 温かいお茶を飲んで、ほっとする時間を作りましょう
- 早く寝ましょう。睡眠が一番の薬です。夜10時には布団に入りましょう

#### 【祈りの言葉】（2-3行）
→ 短くて温かい祝福の言葉

**例:**
```
あなたに幸せな恋が訪れますように。
あなたの優しさが、素敵な人に届きますように。
あなたの笑顔が、誰かの心を温めますように。
きっと大丈夫です。
```

#### 【今日からできること】（1-2行）
→ 今すぐ始められる簡単な行動

**例:**
```
今日は鏡を見て、自分に向かって笑顔を作ってみてください。
それだけで、あなたの魅力が2倍になりますよ。
```

**入力:** 「好きな人に振られて辛いです」

**改善後の応答（抜粋）:**
```

あなたの心は、とても純粋で、深く愛することのできる
素晴らしい光を宿しています。

振られて辛い、その気持ち、痛いほど伝わってきます。
でも、大丈夫。この痛みは、あなたがもっと大きく羽ばたくための、
大切な調整の時間です。

### シャーマンからのアドバイス

・【自分の気持ちを、そっと抱きしめる時間】を作ってください。
  悲しい、悔しい、寂しい、どんな気持ちも、否定しないで、
  「そうだね」と優しく受け止めてあげましょう。

・【小さな喜びを、意識して見つける練習】をしてみてください。
  空の青さ、花の香り、美味しいご飯。
  「ああ、幸せだな」と感じる瞬間を、ノートに書き留めてみましょう。

・【鏡の中の自分に、優しく微笑みかける】時間を持ちましょう。
  「あなたは、大丈夫。あなたは、素晴らしい」と、
  心の中で語りかけてください。

### 祈りの言葉

宇宙の愛よ、この子の心を深く包み込み、癒したまえ。
あなたに、豊かな幸せが訪れますように。

### 今日からできること

今日からできることとして、寝る前に、両手を合わせて、
今日頑張った自分に「ありがとう」と心の中で唱えてみましょう。
```
