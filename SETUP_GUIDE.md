## ⚠️ 重要: Gemini API Keyの設定

現在はテストキーで動作していますが、**実際に使用するには、あなた自身のGemini API Keyが必要です。**

### API Keyの取得と設定手順

1. **Gemini API Keyを取得**
   - [Google AI Studio](https://makersuite.google.com/app/apikey)にアクセス
   - Googleアカウントでログイン
   - 「Get API Key」をクリック
   - 生成されたAPIキーをコピー

2. **.envファイルを編集**
   ```bash
   # .envファイルを開いて、以下のように変更してください
   GEMINI_API_KEY=あなたの実際のAPIキー
   PORT=3000
   NODE_ENV=development
   ```

3. **サーバーを再起動**
   ```bash
   # サーバーを停止して再起動
   npm start
   ```

## 🐛 トラブルシューティング

### サーバーが起動しない
- Node.jsのバージョンを確認（v14以上推奨）
- `npm install`を再実行
- ポート3000が使用中でないか確認

### APIエラーが出る
- Gemini API Keyが正しく設定されているか確認
- APIキーのクォータを確認
- インターネット接続を確認

### フロントエンドが表示されない
- ブラウザのコンソールでエラーを確認
- キャッシュをクリアして再読み込み（Ctrl+Shift+R）
